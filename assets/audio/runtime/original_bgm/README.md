# Original BGM Runtime Baseline

These are unchanged copies of `original/rs_tonyu_0.031/Ogg/1-4.ogg` used as
the temporary runtime soundtrack until replacement music is selected.

- Arcade stages 1-3 play tracks 1-3 at ranks 0, 6, and 11.
- Endless difficulties 1-4 play tracks 1-4 respectively.
- Every stream loops from its beginning and stops on the title screen.
- Runtime gain is applied by `scripts/core/bgm_player.gd`; source files are not
  normalized or otherwise processed.
